# Gearly.com.tr - Frontend Arayüz Projesi

Bu proje, oyun hesapları ve dijital ürün satışı gerçekleştiren Gearly.com.tr platformunun kullanıcı tarafındaki (Client-side) responsive tasarımlarını, modern UI/UX akışlarını ve dinamik entegrasyonlarını barındıran Ön Yüz (Frontend) projesidir.

### 🛠️ Kullanılan Teknolojiler & Özellikler
- **Arayüz Altyapısı:** HTML5, Modern CSS3 (Değişkenler ve Flexbox/Grid mimarisi), Vanilla JavaScript (ES6+).
- **Asenkron Veri İletişimi:** Arka planda çalışan Node.js REST API servisleri ile tam entegre çalışmaktadır.
- **Modern UI/UX Elemanları:** Dinamik SVG tabanlı animasyonlu dünya küresi (Hero Globe Visual), anlık satın alma bildirim simülasyonları, modal/overlay yönetimleri ve akıcı CSS geçiş efektleri.
- **Canlı Destek Entegrasyonu:** Kullanıcıların e-posta doğrulamasıyla anında dahil olabildiği, SSE (Server-Sent Events) tabanlı gerçek zamanlı canlı destek arayüzü.

### ⚙️ Öne Çıkan Mimari Çözümler
- **Decoupled Architecture:** Ön yüz süreçleri, arka yüz (Backend) servislerinden tamamen bağımsız ve modüler bir yapıda geliştirilmiştir.
- **Mobil Uyum (Responsive):** Tamamen medya sorguları (media queries) optimize edilerek mobil, tablet ve masaüstü cihazlar için esnek bir deneyim tasarlanmıştır.
