# Gearly.com.tr - Frontend Projesi

Bu proje, oyun hesapları ve dijital ürün satışı gerçekleştiren Gearly.com.tr e-ticaret platformunun kullanıcı tarafındaki (Client-side) arayüz tasarımlarını, kullanıcı deneyimini (UI/UX) ve dinamik sayfa yönetimini barındıran ön yüz (Frontend) projesidir.

### 🛠️ Kullanılan Teknolojiler
- **Teknoloji Altyapısı:** HTML5, CSS3, JavaScript (Kullandıysan React veya Vue)
- **Tasarım / UI:** Mobil uyumlu (Responsive) tasarım mimarisi.
- **Mimari Yapı:** Ön yüz süreçleri, arka yüz (Backend) servislerinden tamamen bağımsız (Decoupled) şekilde geliştirilmiştir.

### ⚙️ Öne Çıkan Özellikler
- **Dinamik Ürün Listeleme:** Arka plandan ve veri tabanından gelen oyun hesaplarının ve dijital ürünlerin kullanıcıya akıcı bir şekilde sergilenmesi.
- **Kullanıcı Deneyimi (UX):** Satın alma adımları, kategori filtrelemeleri ve hızlı yüklenen arayüz yapısı.
